简介

针对乐视云计算android端播放器sdk4.0的说明文档

sdk所支持的业务如下：

  • 点播业务
  • 直播业务
  • 广告业务

android运行环境

  • android4.0或者以上
  • 当前版本库是armeabi-v7a

android sdk 4.0 api架构

  • MediaPlayer-like架构
  • opengl es 渲染
  • opensl音频播放
  • 支持MediaCodec硬解码
  • 支持全景视频播放
