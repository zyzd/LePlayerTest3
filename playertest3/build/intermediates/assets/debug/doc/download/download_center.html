DownloadCenter API说明文档

方法名 简介
getInstances 获取 DownloadCenter对象，单例模式
downloadVideo 下载视频
allowShowMsg 是否显示下载toast提示
setDownloadSavePath 设置本地视频保存路径
setDownloadRateText 设置当前视频的默认下载码率
getDownloadInfoList 获取所有下载任务列表
getDownloadInfoSize 获取下载任务的数量
setMaxDownloadThread 设置同时下载的最大线程数
resumeDownload 恢复下载
stopDownload 暂停当前视频下载
retryDownload 重试下载失败的视频
stopAllDownload 停止当前所有视频
cancelDownload 取消下载并且删除数据库中的信息,可以删除本地文件
findDownloadInfo 查找当前VU所对应的数据库中的DownloadInfo信息
getDownloadFilePath 获取当前视频保存的本地路径
isDownloadCompleted 查询当前视频是否下载完成
registerDownloadObserver 注册下载状态监听的Observer
unregisterDownloadObserver 取消注册状态监听Observer
setOnDownloadListener 设置下载过程中监听事件，和码率设置搭配使用

/**
 *  下载视频(3.0已有的接口)
 *  
 * @param userKey 用户密钥(可以为“”)
 * @param uu 当前用户的UUID(不可以为空)
 * @param vu 当前视频的VUID(不可以为空)
 */
public void downloadVideo(String userKey, String uu, String vu)

/**
 *  下载视频（3.0已有的接口），已过期，不建议使用这种方式
 * 
 * @param userKey 用户密钥(可以为"")
 * @param uu 当前用户的UUID(不可以为空)
 * @param rateText 需要下载的视频码率
 */
    downloadVideo(String userKey, String uu, String vu,String rateText)
/**
 *  下载视频    (4.0新加的接口)

 */
public void downloadVideo(LeDownloadInfo info)

public void setDownloadRateText(String text)  


示例：
downloadCenter.setOnDownloadListener(new DownloadListener() {

    @Override
    public void onGetVideoInfoRate(LeDownloadInfo info, List<String> rates) {
        if (rates != null && rates.size() > 0) {
            downloadCenter.setDownloadRateText(rates.get(0));
        }
    }
});

//内部接口 DownloadListener

public interface DownloadListener {
    /**
     *  获取当前视频所有码率
     *
     * @param info 当前正在下载的视频
     * @param rates 码率列表(这里，这个码率值和播放的没有统一)
     */
    void onGetVideoInfoRate(LeDownloadInfo info, List<String> rates);
}
