LeDownloadObserver API说明文档

回调状态

这些状态统一在 **LeDownloadInfo.getDownloadState()** 方法中能够拿到
下载状态 说明
DOWLOAD_STATE_NO_DISPATCH 已经添加到下载列表中，还没有调度
DOWLOAD_STATE_DISPATCHING 正在调度中
DOWLOAD_STATE_WAITING 已经调度成功，正在等待下载
DOWLOAD_STATE_DOWNLOADING 正在下载中
DOWLOAD_STATE_STOP 停止下载
DOWLOAD_STATE_SUCCESS 下载成功
DOWLOAD_STATE_NO_PERMISSION 下载失败，视频没有下载权限
DOWLOAD_STATE_URL_REQUEST_FAILED 下载失败，没有获取到下载地址
DOWLOAD_STATE_FAILED 下载失败，其他原因

回调事件

    /**
     * 开始下载
     * 
     * @param info
     */
    public void onDownloadStart(LeDownloadInfo info);

    /**
     * 下载进度更新
     * 
     * @param info 
     */
    public void onDownloadProgress(LeDownloadInfo info);

    /**
     * 下载暂停
     * 
     * @param info 
     */
    public void onDownloadStop(LeDownloadInfo info);

    /**
     * 下载完成
     * 
     * @param info 
     */
    public void onDownloadSuccess(LeDownloadInfo info);

    /**
     * 下载失败
     * 
     * @param info
     * @param msg 失败的原因
     */
    public void onDownloadFailed(LeDownloadInfo info, String msg);

    /**
     * 取消下载
     * 
     * @param info
     */
    public void onDownloadCancel(LeDownloadInfo info);
    /**
     * 开始调度的视频
     *
     * @param info 
     * @param msg 调度的状态
     */
    public void onDownloadInit(LeDownloadInfo info, String msg);
