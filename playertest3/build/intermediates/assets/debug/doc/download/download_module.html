简介

下载模块

下载模块主要和乐视云视频点播业务搭配使用。下载云点播视频到SD卡上，方便用户在断网以及无法使用网络的情况下观看视频。

更新介绍

和3.0相比较：
  1. 性能优化，对网络请求和内存占用以及包大小都进行了优化处理。
  2. 增加了更详细的下载状态描述。
  3. 优化了冗余接口以及不明确接口(注意，3.0升级4.0需要做稍稍的改动)。
  4. 解决了深圳地区已经发现的数据劫持的问题。
  5. 增加了新的接口，方便用户直接对接，不再需要进行数据封装。
