初始化乐视云服务

在Application中初始化乐视云服务，保证程序开启的时候，服务已经成功初始化完毕。

public class MApplication extends Application {


    public static String getProcessName(Context cxt, int pid) {
        ActivityManager am = (ActivityManager) cxt.getSystemService(Context.ACTIVITY_SERVICE);
        List<RunningAppProcessInfo> runningApps = am.getRunningAppProcesses();
        if (runningApps != null) {
            for (RunningAppProcessInfo procInfo : runningApps) {
                if (procInfo.pid == pid) {
                    return procInfo.processName;
                }
            }
        }
        return null;
    }

    @Override
    public void onCreate() {
        super.onCreate();
        String processName = getProcessName(this, android.os.Process.myPid());
        if (getApplicationInfo().packageName.equals(processName)) {
            //TODO CrashHandler是一个抓取崩溃log的工具类（可选）
            CrashHandler.getInstance(this);
            LeCloudProxy.init(getApplicationContext());
            LeCloudPlayerConfig.getInstance().setDeveloperMode(true).setIsApp();
        }
    }
}

创建播放器

创建PlayContext

playContext是播放器的上下文，用于保存播放器的临时数据和上下文

    /**
     * 初始化PlayContext PlayContext作为播放器的上下文而存在，既保存了播放器运行时的临时数据，也记录播放器所需要的环境变量。
     */
    private void initPlayContext() {
        playContext = new PlayContext(this);
        //当前视频渲染所使用的View，可能是Surfaceview(glSurfaceView也属于Surfaceview ),也可能是textureView
        playContext.setVideoContentView(videoView);
    }

创建letvplayer对象，通过setPlayContext关联playContext,同时对播放器进行初始化操作。 如果此处SurfaceView已经创建完毕，可以通过setDisplay将surface设置到player中

关于setDisplay()

SurfaceView创建的时候，是一个异步过程，所以此时如果SurfaceView的onSurfaceCreated()没有回调的话，surface可能是null

setDisplay可以延迟到ISplayer.MEDIA_EVENT_PREPARE_COMPLETE事件收到后，start之前设置

    player = new LetvPlayer();
    player.setPlayContext(playContext); //关联playContext
    player.init();
    if (surface != null) {
        player.setDisplay(surface);
    }

乐视云参数设置

乐视云点播参数设置

通过setParameter方法，将点播所需要的参数设置一下，播放器会通过这些参数请求点播相关业务

    Bundle bundle=new Bundle();
    bundle.putInt(PlayProxy.PLAY_MODE, EventPlayProxy.PLAYER_VOD);// 点播业务
    bundle.putString(PlayProxy.PLAY_UUID, uu);
    bundle.putString(PlayProxy.PLAY_VUID, vu);
    bundle.putString(PlayProxy.PLAY_USERKEY, userKey);
    // bundle.putString(PlayProxy.PLAY_SERVICE_NUMBER, "业务线id");
    // bundle.putString(PlayProxy.PLAY_CUSTOMERID, "用户id");
    player.setParameter(player.getPlayerId(), bundle);

乐视云直播参数设置

通过setParameter方法，将直播所需要的参数设置一下，播放器会通过这些参数请求直播相关业务

传入liveid或者是streamid

LiveID举例：201504213000012

StreamID举例：20150421300001210

如果之前有对接过sdk3.0的开发者，以前使用了streamid是类似这样：A2015111600547，那么在sdk4.0中，将会变成乐视云活动直播参数，请看下一节的参数设置

    Bundle bundle=new Bundle();
    bundle.putInt(PlayProxy.PLAY_MODE, EventPlayProxy.PLAYER_LIVE);// 直播业务
    bundle.putString(PlayProxy.PLAY_LIVEID, liveId);
    //bundle.putString(PlayProxy.PLAY_STREAMID,streamId);//streamId或者liveId,二选一

    //bundle.putBoolean(PlayProxy.PLAY_ISRTMP,isRtmp);//isRtmp或者isHls，二选一
    bundle.putBoolean(PlayProxy.PLAY_USEHLS, isHls);

    //bundle.putBoolean(PlayProxy.PLAY_ISLETV, false);//用于标注是否是乐视网的视频
    player.setParameter(player.getPlayerId(), bundle);

监听播放器和播放器准备

设置播放器监听,通过监听接口，播放器会主动得给调用者发送一系列的事件。 你可以在这里处理播放器异常，视频播放完毕等事件。

关于播放器事件,请查看LePlayer播放器Api说明

    player.setOnPlayStateListener(new OnPlayStateListener() {
            @Override
            public void videoState(int state, Bundle bundle) {
                if (player == null) {
                    return;
                }
                switch (state) {
                case ISplayer.MEDIA_EVENT_PREPARE_COMPLETE:
                    player.start();
                    break;
                ......
                }
            }
        });

播放器开始准备，这时候开始进行验证和链接媒体服务器，当准备完成的时候，播放器会监听到监听完成事件：ISplayer.MEDIA_EVENT_PREPARE_COMPLETE。此时可以执行start()

    //if (lastPosition > 0) {
    //    player.seekTo(lastPosition);

    player.prepareAsync();

如何实现继续播放??

1) 在stop方法之前获取到退出时的最后播放时间

   lastPosition=player.getCurrentPosition();
   player.stop();
   ......

2) 在prepareAsync之前，将上次播放通过seekTo方法设置

    if (lastPosition > 0) {
        player.seekTo(lastPosition);
    }
    player.prepareAsync();

销毁播放器

当需要结束播放的时候，需要销毁播放器资源

private void stopPlayer(){
    if(player!=null){
        player.stop();
        player.release();
        player=null;
    }
}
