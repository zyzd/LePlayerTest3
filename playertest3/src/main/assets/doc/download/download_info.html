LeDownloadInfo API 说明文档

LeDownloadInfo是SDK下载实例。所以LeDownloadInfo中只是在一种情况下需要用户主动去修改。其他情况下，用户都只能用于读取信息。

直接调用downloadVideo(LeDownloadInfo info )去下载视频，这也是当前所推荐的调用方式

    LeDownloadInfo info = new LeDownloadInfo();
    info.setUu(UUID); // 必填,否则不能下载视频
    info.setVu(VUID); // 必填，否则不能下载视频
    info.setUserKey(UserKey);//用户信息UserKey，选填
    info.setCheckCode(CheckCode);// 用户信息CheckCode,选填
    info.setPayerName(PayerName);// 用户信息PayerName，选填
    //扩展数据，这次字段提供给用户可扩展使用。这些信息在调度过程中都不会做任何的修改,只是单独保存到数据库中。
    info.setString1(String);// 选填
    info.setString2(String);// 选填
    info.setString3(String);// 选填
    info.setString4(String);// 选填
    info.setInt1(Int);// 选填
    info.setInt2(Int);// 选填

    DownloadCenter.downloadVideo(info);

需要用到的方法

方法名称 说明
getDownloadState 获取当前视频的下载状态，注意不能主动修改
getFileName 获取当前视频文件的名称
getFileSavePath 获取当前视频文件保存的路径
getProgress 获取当前视频的下载进度
getFileLength 获取当前视频文件的大小
getRateText 获取当前数据库中保存的视频所对应的码率
