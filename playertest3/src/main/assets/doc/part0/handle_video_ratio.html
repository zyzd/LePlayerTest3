处理视频比例

如果幸运的话，你的播放器已经可以正常播放了，但遗憾的是，视频的比例显示是不对的，此时需要获取到视频的比例，然后对显示的view设置一下比例。

处理 MEDIA_EVENT_VIDEO_SIZE 回调事件

视频播放第一帧画面之前，会收到MEDIA_EVENT_VIDEO_SIZE的回调事件,此时可以获取到视频的宽高。


    private int width;
    private int height;
    player.setOnPlayStateListener(new OnPlayStateListener() {
            @Override
            public void videoState(int state, Bundle bundle) {
                if (player == null) {
                    return;
                }
                switch (state) {

                case ISplayer.MEDIA_EVENT_VIDEO_SIZE://获取到视频宽高
                    width=player.getVideoWidth();
                    height=player.getVideoHeight();

                    //1.算出视频比例
                    float ratio=(float)width/height;

                    //2.获取到当前显示view的宽高
                    //3.重新计算宽高
                    //4.重新设置layoutParams
                    mView.setLayoutParams();

                    break;
                ......
                }
            }
        });

处理SurfaceView 的surfaceChanged()方法

如果使用了SurfaceView 显示视频内容时候，当surfaceview的宽高变化的时候，会触发的surfaceChanged()方法，为了保证视频的比例，应该告诉播放器此时显示的view已经改变了大小。

    surfaceView.getHolder().addCallback(new Callback() {

            @Override
            public void surfaceDestroyed(SurfaceHolder holder) {
                ......
            }

            @Override
            public void surfaceCreated(SurfaceHolder holder) {
                ......
            }

            @Override
            public void surfaceChanged(SurfaceHolder holder, int format, int width, int height) {

                if(player!=null){
                     PlayerHelper.setViewSizeChange(player, width, height);
                }
