LetvPlayer播放器API说明

在一个视频播放的过程中，播放器需要经历创建、播放、最后释放的过程。播放过程期间，播放器会主动的告诉我们一些回调事件,或者说我们需要主动的获取一些播放器参数，这时候就需要充分利用好LePlayer所提供的方法了！

初始化和准备

方法名字 简要说明
init 播放器初始化
setParameter 设置播放器参数
setDataSource 设置播放源
setOnPlayStateListener 设置播放器状态监听
setDisplay 设置显示窗口
prepareAsync 播放器开始准备
setVolume 设置播放器音量
方法名字 简要说明
setPlayContext 设置播放器运行时上下文

停止和释放

方法名字 简要说明
reset 播放器状态重置
stop 停止播放
release 释放播放器对象

播放中操作

方法名字 简要说明
seekTo 快进或者快退,参数单位：毫秒
pause 暂停
isPlaying 是否正在播放
start 开始播放
方法名字 简要说明
playedByDefination 切换码率,参数设置对应((PlayContext)getPlayContext()).getDefinationsMap()中的key值
resetPlay 重新播放
isComplete 是否播放完毕
getPlayContext 获取当前播放器上下文
getStatus 主动获取当前播放器所处的状态

播放中获取参数

方法名字 简要说明
getVideoWidth 获取视频宽
getVideoHeight 获取视频高
getCurrentPosition 获取当前播放时间,单位：毫秒
getBufferPercentage 获取当前缓冲进度
getDuration 获取视频播放时长
getPlayerId 获取播放器id,id可以作为该播放器的唯一标识
getAttribute 获取播放器一些附加属性
getObserver 获取播放器的一个观察者
getStatus 获取播放器状态，返回事件见播放器状态事件

回调事件

在设置了setOnPlayStateListener(OnPlayStateListener playStateListener)之后，播放器返回的事件

    public interface OnPlayStateListener {
    /**
     * @param state 表示播放器返回的事件
     * @param bundle 对于某些特定的事件返回时，会携带相关的数据，这时可以从bundle中获取到
     */
    public void videoState(int state,Bundle bundle);
    }

播放器状态回调事件

事件名字 简单说明
MEDIA_EVENT_VIDEO_SIZE 播放器获取到视频宽高
MEDIA_EVENT_PLAY_COMPLETE 播放视频完毕
MEDIA_EVENT_FIRST_RENDER 第一帧视频渲染完成
MEDIA_EVENT_SEEK_COMPLETE seek完成
MEDIA_EVENT_PREPARE_COMPLETE 播放器准备完成
MEDIA_EVENT_BUFFER_START 播放器开始缓冲
MEDIA_EVENT_BUFFER_END 播放器缓冲结束
MEDIA_ERROR_NO_STREAM 播放错误，连接媒体服务器错误或者超时
MEDIA_ERROR_PREPARE_ERROR 播放错误，连接媒体服务器错误或者超时
MEDIA_ERROR_DECODE_ERROR 解码错误

MEDIA_EVENT_VIDEO_SIZE

当收到该事件之后，可以获取到视频的宽和高数据

case ISplayer.MEDIA_EVENT_VIDEO_SIZE:
            if (videoView != null && player != null) {
                /**
                 * 获取到视频的宽高的时候，此时可以通过视频的宽高计算出比例，进而设置视频view的显示大小。
                 * 如果不按照视频的比例进行显示的话，(以surfaceView为例子)内容会填充整个surfaceView。
                 * 意味着你的surfaceView显示的内容有可能是拉伸的
                 */
                videoView.onVideoSizeChange(player.getVideoWidth(), player.getVideoHeight());
                /**
                 * 获取宽高的另外一种方式
                 */
                bundle.getInt("width");
                bundle.getInt("height");
            }
break;
事件名字 简单说明
PLAYER_PROXY_ERROR 播放代理错误

PLAYER_PROXY_ERROR

该事件收到之后，表明播放器业务层出错了，可以获取到具体的错误信息

case ISplayer.PLAYER_PROXY_ERROR:
    int errorCode = bundle.getInt("errorCode");
    String msg = bundle.getString("errorMsg");
break;

视频信息相关事件

事件名字 简单说明
EventPlayProxy.PROXY_VIDEO_INFO_REFRESH 获取到视频的基本信息，比如标题，是否支持下载等
EventPlayProxy.PROXY_WAITING_SELECT_DEFNITION_PLAY 获取到视频所支持的码率
EventPlayProxy.PROXY_WATING_SELECT_ACTION_LIVE_PLAY 获取到活动直播信息

EventPlayProxy.PROXY_VIDEO_INFO_REFRESH

该事件收到后，可以获取到视频的基本信息

case EventPlayProxy.PROXY_VIDEO_INFO_REFRESH:
            if (player == null || player.getPlayContext() == null) {
                return;
            }
            ((PlayContext) player.getPlayContext()).getVideoTitle();
            ((PlayContext) player.getPlayContext()).isCanbeDownload();
            break;
        }

EventPlayProxy.PROXY_WAITING_SELECT_DEFNITION_PLAY

收到该事件之后，可以获取到视频对于的码率

case EventPlayProxy.PROXY_WAITING_SELECT_DEFNITION_PLAY:
            PlayContext playContext = (PlayContext) player.getPlayContext();
            if (playContext != null) {
                playContext.getDefinationsMap();
            }
            break;

EventPlayProxy.PROXY_WATING_SELECT_ACTION_LIVE_PLAY

收到该事件之后，可以获取到活动直播对于的活动信息

case EventPlayProxy.PROXY_WATING_SELECT_ACTION_LIVE_PLAY:
            PlayContext playContextAction = (PlayContext) player.getPlayContext();
            ActionInfo actionInfo = playContextAction.getActionInfo();//获取到活动直播信息
            if (actionInfo != null) {
                /**
                 * 如果当前活动直播是多个直播窗口， 选择一路可用的活动直播
                 */
                LiveInfo liveInfo = actionInfo.getFirstOnUseOrSeekLive();
                if (liveInfo != null) {
                    mPlayContext.setLiveId(liveInfo.getLiveId());
                }
            }
        break;
